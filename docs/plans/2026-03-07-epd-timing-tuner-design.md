# EPD 时序调节测试页面设计

## 目标

改造 EpdTestViewController，增加 Fast GL16 波形时序的实时调节功能，以及"白 DU → GL16"全屏切换测试。

## 功能需求

### 1. Fast GL16 时序调节
- 3 个预设方案（Fast/Normal/Slow）
- 15 个 phase 分 3 组（0-4, 5-9, 10-14），每组 5 个 +/- 步进按钮
- 步进 ±5μs，范围 5~500μs
- 点击刷新按钮时下发时序到 driver

### 2. 白 DU → GL16 全屏切换
- 一个按钮执行：DU 刷白 → GL16 显示目标内容
- 测试是否能消除 GL16 闪黑现象

### 3. 测试图案增强
- 保留 A/B 切换（灰度渐变 + 棋盘格/条纹）
- 增加中文正文样例文字

## 新增 epd_driver 接口

```c
void epd_driver_set_fast_gl16_times(const int times[15]);
const int* epd_driver_get_fast_gl16_times(void);
esp_err_t epd_driver_white_du_then_gl16(void);
```

## 页面布局

```
┌─────────────────────────────┐
│ [←] EPD Refresh Test        │ Header
├─────────────────────────────┤
│ [Fast] [Normal] [Slow]      │ 预设选择行
│ [0-4] [5-9] [10-14]         │ Phase 分组选择
│  P0: 10  [+][-]             │ 当前组 5 个 phase
│  P1: 10  [+][-]             │
│  P2: 10  [+][-]             │
│  P3: 10  [+][-]             │
│  P4: 10  [+][-]             │
├─────────────────────────────┤
│ [FastGL16] [DU→GL] [GL16] [Clear] │ 刷新按钮
├─────────────────────────────┤
│  灰度渐变区域               │
│  正文样例文字               │ 测试图案 (A/B)
│  棋盘格/条纹区域            │
└─────────────────────────────┘
```

## 预设时序值

| 预设   | 值                                          | 总时长 |
|--------|---------------------------------------------|--------|
| Fast   | 10,10,10,10,10,15,15,20,20,30,30,50,50,80,120 | 500μs |
| Normal | 15,15,15,15,15,20,20,30,30,40,40,60,60,100,150 | 625μs |
| Slow   | 20,20,20,20,20,30,30,40,40,60,60,80,80,120,200 | 840μs |

## 数据流

- VC 内存维护 `int currentTimes_[15]`
- +/- 按钮修改内存值并更新 TextLabel 显示
- 点击刷新按钮时：`epd_driver_set_fast_gl16_times()` → 切换图案 → 绘制 → 刷新

## 白 DU → GL16 实现

1. 保存当前帧缓冲区到临时 buffer
2. `memset(framebuffer, 0xFF)` 填白
3. `epd_hl_update_screen(MODE_DU)` 快速刷白
4. 恢复帧缓冲区内容
5. `epd_hl_update_screen(MODE_GL16)` 显示目标
